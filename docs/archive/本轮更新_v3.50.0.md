## 这轮新做的（v3.50.0）

### 1. 版本号改细 ✅（你说太随便）
改成 **v主.次.修** 三段：主=产品大版本(3)、次=较大功能轮、修=小修小补。小改动走最后一位，不再一轮就跳一个大版本。**当前 v3.50.0**。

### 2. 阅读进度条可直接拖/点跳读 ✅
顶部那条进度条现在能**直接点或按住拖**到任意百分比松手就跳过去（悬停/拖动时会冒出一个圆点把手）。翻页和滚动两种模式都支持。

### 3. 「生成书单」挪到阅读记录里 ✅
不再放在多选栏。现在在**「阅读历程 → 总览」**有「生成书单」入口，用你**读过/在读**的书直接排成书单（带简评，可选文字或图片）。多选栏只留收藏/打标签/置顶作者/删除/退出。

### 4. 种树也按最新阅读排前 ✅
阅读记录里的「读书统计/种树」现在和书架一样——**最近读的排最前面**（按 last_open 排序）。

### 5. 每本书的「阅读回顾」（第二遍/第三遍的时间）✅
书详情新增一行**「阅读回顾」**：显示这本书你**读过几天、估算读了几遍**，点开看**时间线**——每次阅读的日期、读了多久、读到百分之几，并标出「第 N 遍开始」。
- 遍数是估算：相邻阅读日间隔超过 10 天、或进度明显回退（又从头读）就算新一遍。逻辑已单测。
- 数据来自每天的阅读记录（reading_diary），不额外占库。读得越多，这里越完整。
