# 更新说明 v3.55.0

## 笔记/划线（按你说的重做了）
- 划线列表默认只显示这句话 + 「编辑 / 删除」，**不再一上来就摆一排改色点**。
- 点「编辑」才在**这条下面展开一个笔记框**：框会**随你输入逐渐变高**，旁边带**改色**点，写完点「完成」。**不再弹那个白色系统框**了。
- 划线/笔记的文字**点一下就跳到正文那个位置**。
- 选中正文点「笔记」也是直接进这个内嵌编辑框（不再 prompt）。

## 翻页 / 长按
- **长按不再误翻页**：现在只要这次触摸里发生过选字、或按住超过 0.45 秒、或手指移动过，都不翻页——专心做笔记不会被翻走。
- 翻页「跳页」：加严了点击识别（移动/选字都不触发翻页），减少误触连翻。
- 翻页动画没删：宽屏用真实翻书动画，手机窄屏用单页滑入动画（左右都有）。如果你平板上看着像没动画，多半是被判成了宽屏的真实翻页或卡顿，下次发我平板宽度我再调。

## 章节认数字了
《旧故新长》这种**用 1、2、3 纯数字分章**的，现在目录和「理清章节」都能认出来了（要求至少 3 个、且大体递增，避免把零散数字误判成章节）。已单测：纯数字分 4 章能认出、散文里的零散数字不会误判、第X章照常。

## 找回简评（这次能看清到底恢复没恢复）
之前「执行恢复」可能写了 0 行却报成功——这版改成**按真正写入的行数计数**，并**列出到底恢复了哪几本**；还加了等锁，避免库被占用时悄悄失败。
你说第一次扫到 3 条、执行却没恢复：再点一次「执行恢复」，看它现在显示「已恢复 N 条：书名…」还是「没有写入（已经有简评/没匹配上）」。
- 若显示「已恢复…旧故新长」→ 回书房 Ctrl+Shift+R 强刷就能看到。
- 若显示「没有写入…已经有简评」→ 说明库里这本其实已经有简评了，是显示没刷新，强刷即可。
- 都不是的话，把这本书名单独发我，我直接给你从备份里把那条捞出来。
