# 更新说明 v3.56.0

## 改了（你这次提的）
1. **选中的书批量标「已读 / 在读」**：多选栏新增「标已读」「标在读」两个按钮——选中那两本（或多本）点一下就改状态，不用一本本进详情切了。（拖到书架分区改状态实现起来不稳，先用按钮，效果一样。）
2. **收藏「这一本」最上面放总评**：进收藏翻到某本，最上面直接显示这本书的「我的总评」，下面才是划线/收藏的句子。（不是只放在阅读历程总览顶上。）
3. **收藏里加了搜索栏**：顶部有搜索框，输书名/作者就跳到对应那一本，书多时好定位。
4. **跑维护任务时搜索不再卡死**：给数据库读操作加了等锁(8秒)——维护任务在写库时，书房搜索会等一下而不是直接失败。
5. **「完结识别 + 分章节」可以停了**：加了「停止」按钮（这任务本来就支持断点续跑，停了下次点"继续扫描"接着来）。顺手让所有任务的停止按钮都能用了。
