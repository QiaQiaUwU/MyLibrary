# 更新说明 v3.57.0（翻页 + 切页）

先跟你道个歉：上轮我猜你在滚动模式，是我没认真看你给的翻页截图，对不起。这轮按你说的两点正经修了。

## 1. 翻页动画回来了（而且更可靠）
查到根因：原来的翻页动画用的是一个**从外网 CDN(unpkg) 加载的翻页库**。你是在**局域网/本地**打开的（192.168.x.x），这个外网库**根本下载不下来**，于是就退化成"直接瞬间跳页、没有动画"——这就是你说的"一点都没有了"。
现在**彻底不用那个外网库**，改成内置的"单页滑入"翻页：离线也能用、各设备一致、左右翻都有滑动动画（带前缘阴影，更像翻纸）。顺手把那个连不上的外网脚本删了，阅读器加载也快一点。

## 2. 切页不再把字/句子切一半（按屏幕自动分页）
真正的毛病在分页算法：原来是**按固定字数切页**——可中文有对话、空行、长短段，同样的字数渲染出来高矮差很多，一页塞多了就**超出屏幕、底下那行被切掉一半**。
现在改成**量着真实高度分页**：拿你这块屏幕真实的页面盒子高度，往里一行行填字，填到快超出这一页就断页，并尽量**断在换行处**（不把句子/字切断）。所以：
- 任何屏幕尺寸都自动贴合，不会再切一半；
- **横竖屏切换/改窗口大小**会自动重新分页（加了监听）；
- 改字号/行距也会重新按高度排。
