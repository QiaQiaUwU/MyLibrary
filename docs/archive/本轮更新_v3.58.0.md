# 更新说明 v3.58.0（跨页做笔记）

## 跨页划线/做笔记（新）
翻页模式下，一句话/一段被切到两页时，没法一次选完——现在能跨页划了：
1. 在**起点**那页选中开头的文字，选择菜单点 **「跨页」**；
2. 顶部出现「跨页划线中」提示，**翻到结尾那页**；
3. 在**结尾**处选中文字，点 **「划到这里」**——从起点到结尾整段（哪怕跨好几页）一起划上，可继续写笔记/改色。
4. 中途不想划了，点顶部那条提示就取消。
（跨页的划线在每一页上都会正确显示对应的那一截。）
