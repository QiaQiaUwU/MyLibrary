# 更新说明 v3.59.0（跨页做笔记改成"拖着往下续" + 翻页动画说明）

## 跨页做笔记：按你说的改了
- **删掉**了上一版那个"跨页/划到这里"两步按钮（确实别扭）。
- 现在用最自然的方式：**滚动模式**下，按住选中、手指往下拉到屏幕底部，**会自动往下滚**，选择就一直续下去，跨多少屏都行——不用任何按钮。松手就停。
- 为什么放在滚动模式：翻页模式是一页一页独立的，系统的选区没法跨过"翻页"这个动作活下来（翻页时整页会重画，选区就断了）。所以跨页连续划线天然属于滚动模式。读长段、想跨页划的时候，切到「外观 → 滚动」就顺了。

## 翻页动画：我参考了别的阅读器，说一下到底是什么
你不放心"滑入"，我说清楚——我做的这个是 **"覆盖式"翻页**：旧页留在底下不动，新页从一侧滑进来盖上去。这正是 **微信读书**里那个「覆盖」模式（它还有「平移」也是类似滑动）。主流 App 里翻页大致就三类：
1. **仿真/卷曲**（像翻真书那样卷一下）——iBooks、部分版本微信读书有。**问题**：必须加载一个翻页 JS 库，而你是局域网/离线打开的，那个库（之前挂在外网 unpkg）**根本下不下来**，于是退化成瞬间跳页——这就是你之前说"动画一点没有了"的真正原因。
2. **覆盖/平移**（新页滑进来盖住旧页）——纯 CSS、不依赖任何外网库、离线稳。**我现在用的就是这个。**
3. **无动画**（直接换页）。

这版我把覆盖式修对了：之前底下垫的是"下一页"导致滑动时底下闪错内容，现在底下垫的是**被翻走的那一页**，滑动看着就对了，带一点前缘阴影。

**给你三个选择**（你定，我下轮按你要的来）：
- A. 就用现在这个**覆盖滑入**（推荐，离线最稳）；
- B. 想要**真·卷曲翻书**那种——我可以把那个翻页库**打包进项目本地**（不走外网），离线也能用，代价是包大一点、低端设备可能略卡；
- C. 加一个**淡入**或**无动画**选项放设置里，你自己切。
