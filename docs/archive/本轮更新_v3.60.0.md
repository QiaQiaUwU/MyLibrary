# 更新说明 v3.60.0（翻页改成立体翻折 + 可选真·卷曲）

## 关于"仿真卷曲"——先说实话
你说项目一开始有卷曲翻页、后来没了——对，是我前面为了修"局域网/离线下翻页瞬间跳页"，把那个**挂在外网的卷曲翻页库**整个去掉了（它在你这种离线环境下根本下不下来）。这是我处理得不好。
**真·软纸卷曲必须靠那个 JS 库**，而我这边没有网络、也没法把它下载下来打包给你。所以这版我做了两件事：

### 1) 内置「CSS 3D 立体翻页」——离线就有，不再是平移滑入
不靠任何外网库。翻页时，**整页绕着书脊（左缘）翻折转走，露出底下的下一页**，带阴影和透视，是立体的翻页，不是平移。点击两侧翻、左右滑动翻都走这个效果（滑动是跟手拖动那一页）。这就是离线能稳定拿到的最接近真书的效果。

### 2) 想要真·软纸卷曲：放一个文件就启用（离线也行）
我留好了本地接口。你只要把这个文件放到项目的 `frontend/vendor/` 文件夹里：
  **page-flip.browser.js**
（用任意能上网的设备，从 `https://unpkg.com/page-flip@2.0.7/dist/js/page-flip.browser.js` 下载，拷进去即可，详见该文件夹里的 README.txt）
放好后重启服务、阅读器关掉重开、Ctrl+Shift+R——阅读器会**自动检测到它并启用真·卷曲翻页**；没放就用上面的内置 CSS 3D 翻页，绝不报错。
（这条是离线最稳的"既要又要"：默认有立体翻页，想要软卷曲就加一个文件。）

## 翻页模式下跨页做笔记——回答你的提议
你问"翻页模式跨页标记能不能往后滚一点点"。能，但那等于把翻页模式临时变成小滚动，容易和系统选字打架、不太稳。所以我建议：**要跨页连续划线，就切到「外观 → 滚动」模式**——上一版已经做了：滚动模式下按住选中往下拉到屏幕底会自动续滚，跨多少屏都行，松手停，最顺。
如果你坚持要在翻页模式里也能"选到底自动露出下一页续选"，告诉我，我下轮专门做这个（会做成：选字拖到底→把下一页接到当前页下面、可滚动续选→松手成跨页划线）。
