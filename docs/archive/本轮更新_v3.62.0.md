## 翻页"盖住字 / 往前翻不了 / 屏幕适配"——这次找到根因了
你电脑上看到的是**两页并排的横版书**(StPageFlip 在宽屏上自动摊成左右两页)，这带来两个真问题：
- **盖住字**：更要命的是**分页算宽度时按整个窗口算的**，可你的书页其实只有 680px 居中——于是每行被当成能放更多字，排进去就超出真实页宽、底/边被切。**已修：分页严格按真实页宽(680)算，不再按整窗。**
- **往前翻不了 / 翻页怪**：横版两页时左侧几乎贴满文字，点左边要么点到字(变选字)要么被那一页吃掉，没有干净的"上一页"区。**已改成强制单页竖版**(把书宽上限从 1000 降到 720)，宽屏上书居中、两侧留出干净的点击区，点左边＝上一页、点右边＝下一页，不再摊成两页盖字。

请重点试：①电脑上翻页还盖不盖字、最后一行切不切；②点书页**左边空白处**能不能往前翻。要是还翻不回去，告诉我你是点哪、有没有反应，我接着查。
