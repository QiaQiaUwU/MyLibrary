# 更新说明 v3.62.2（找回总评：现在也会查"同名副本"）

你说《仙山深处有魔修》你确实写过总评，让我去库里捞。我这边连不到你电脑上的 library.db，但想明白最可能的去处了：

**你那条总评，多半还在你库里——只是写在了这本书的另一个"重复副本"上。**
你库里有大量重复书。如果你当初在某一个副本上写了总评，而现在点开的是同名的**另一个**副本（空的），看着就像"丢了"。

## 改了什么
「找回丢失的简评」现在**两头找**：
1. 扫历次备份（和上一版新增的每日自动备份）；
2. **新增：查同名书的其它副本**——把"有总评那一份"的总评，补给同名但为空的那几本（不依赖任何备份，纯在你现有库里找）。

已单测：同名两本、一本有总评一本空，能把总评正确补给空的那本；只有一本的不动。

## 你现在这样做
管理 →「找回丢失的简评」→ 点**「扫描可找回的」**：
- 如果显示找到了、列表里有《仙山深处有魔修》→ 点**「执行恢复」**，回去强刷就能看到总评回来了；
- 如果还是 0：说明这条既不在备份、同名其它副本上也没有——那它的来源记录可能已经被删掉了（比如去重时删的、且当时没存备份）。这种我也没法凭空变出来，但**从这版起每天自动备份**，以后不会再这样。
