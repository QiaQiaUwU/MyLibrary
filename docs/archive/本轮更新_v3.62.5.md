# 更新说明 v3.62.5

太好了，能打开了。这版两件事：

## 1. 「列出全部总评」不再"悄悄去重"，每条标出来源
你担心去重把简评省略了——说清楚：原来只合并**书名+简评全文一字不差**的重复，**和作者无关、内容不同的绝不合并**。但既然你说别去重，这版改成：
- 每条简评下面标**来源**（当前库 / 某份备份 / 去重前备份…）；
- 同一条简评在多处出现，只显示一条、但把**所有来源都列出来**给你看，让你确认没漏；
- 内容只要有一点不同，就当两条分开列。
所以你看到的条数 = 当前库+所有备份里**全部不同的简评**。仙山那条没出现，是它确实不在当前库、也不在现有的这几份备份里（多半是写过的那份记录在某次去重时被删、且早于这几份备份）——不是被去重藏起来了。

## 2. 收藏翻页做成"柔软翻页"了
收藏里翻上一本/下一本，现在是**像翻纸一样**：当前页以书脊为轴卷走、正面背对你时换内容、再卷回来，左右方向不同（往后/往前卷的方向相反），带一道翻页阴影。左右滑动或点两边箭头都触发。

> 阅读器正文那边本来就用的是真正的卷曲翻页（StPageFlip，联网能加载到那个库时）；收藏这边因为每"页"是可滚动的长卡片，做的是上面这种柔和的卷页过渡。
