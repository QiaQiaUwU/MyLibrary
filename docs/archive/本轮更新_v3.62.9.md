# 更新说明 v3.62.9（阅读器：进度拖动 / Quill 叉能点了 / 点划线改色）

接着上一版的"先确认新代码进来了"——这版做你列的几个阅读器功能。先做了三个最实在的：

## 1. 进度条：现在看得见、随手能拖
进度拖动其实一直在顶部，但只有一条很细的线、而且工具栏一出来就被禁用了，所以你"找不到"。改了：
- **呼出菜单（点屏幕中间）后，进度条移到标题栏正下方、变粗、小圆点常显**——一眼能看见，按住左右拖就跳到任意位置，松手到位；
- 工具栏收起时，顶部那条细线也照样能拖（快速定位）。

## 2. Quill 的叉能点了
你说得对：Quill 抽屉弹出来时，右上角的关闭叉**正好压在顶部进度条下面**，点不到。原因是进度条层级比抽屉高。这版把 Quill / 列表抽屉的层级提到进度条之上，**叉就能正常点关**了。
（"把 Quill 拉成可拖动小窗"这个改造更大，排在后面单独做，先把点不到叉这个挡路的解决。）

## 3. 点正文里的划线 → 就地改颜色 / 写笔记 / 删除
参考你截图里 Moon+ 那种就地操作：现在**在正文里点一下某条划线**，会在它上方弹个小菜单——
- 一排颜色点，点一下**直接换色**；
- 「笔记」写/改这条的笔记；「删除」删掉这条划线。
不用再专门进侧栏列表去操作了。
