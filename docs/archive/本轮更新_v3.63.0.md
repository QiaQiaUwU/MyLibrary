# 更新说明 v3.63.0（Quill 变成可拖动小窗）

接着做你列的：这版把 **Quill 改成了可拖动的小窗**。

## Quill 小窗
- 不再是占满整条右边的大抽屉，改成**右下角一个圆角小窗**（约 400 宽、屏高的 64%）；
- **按住它的标题栏（"Quill"那一条）就能拖**，想放哪放哪——挪开正文、挪开进度条都行；标题中间有个"⋮⋮"提示可拖；
- 拖到的位置会记住；关闭的叉、历史、新对话都照常（点这些不会误触发拖动）。
