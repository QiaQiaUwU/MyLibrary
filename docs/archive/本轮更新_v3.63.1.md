# 更新说明 v3.63.1（划线调长度 + 分页边距调舒服）

把剩下两件做了：

## 1. 划线可以调长度了
在正文点一下某条划线弹出的小菜单里，多了一排：**头扩 / 头缩 / 尾缩 / 尾扩**。
- 「尾扩」把划线往后多吃一句（到下一个标点）、「尾缩」往回收一句；「头扩/头缩」同理调起点；
- 可以连点，连续扩/缩，菜单不会关；
- 改完即时重画、并存盘（数据库里存的是这条划线盖住的文字，长度=文字长度）。

> 我没用"拖小handle"那种，是因为拖拽要在正文里精确点字符、容易飘；按标点一句一句扩缩更稳、也更符合中文断句。你要是更想要"拖着拉"，告诉我，我再加。

后端也放开了：划线的 PATCH 接口现在支持改 position / text（之前只让改颜色和笔记）。

## 2. 分页边距调舒服一点
正文左右**留白加宽了**（28→34），不再贴着边；翻页和滚动都一样。行距/字号你本来就能在「外观」里自己调。
（这块比较主观——你要还嫌挤/嫌空，告诉我"边再宽点 / 行再松点 / 字再大点"，我对着你那台再调。）
