# 更新说明 v3.63.2（按 Moon+ 把分页排版改舒服）

上一版我只是把边距调宽，没改到点子上。这版按你给的 Moon+ 截图，把真正让它"舒服"的两点做了：

## 段落排版（关键）
- **段与段之间留白**：以前是把整段文字连续排（段间靠原文换行，挤），现在每个自然段单独成段、**段间有空隙**，像 Moon+ 那样一段一段清清爽爽；
- **首行缩进**沿用原文的全角空格（中文小说基本都有），首行缩进、回行顶格，符合中文排版习惯；
- 连续空行会被合并成**统一的段间距**，不会这本松那本紧。

这套段落排版**翻页和滚动两种模式都用**；而且我把"分页测高"也改成按同样的段落结构来量，所以**每页不会因为多了段间距而把底部文字切掉**（测高偏保守，宁可底部多留一点空，也不切字）。划线、划线调长度、改色这些都照常工作（段落里字符定位是精确保留的，已测）。

## 还想再调的话
段间距、行距、字号都还能再动。你看了实际效果后，要更松/更紧告诉我一个方向（比如"段间距再大一点"或"行距收一点"），我直接调数值。
