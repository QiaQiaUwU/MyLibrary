# 更新说明 v3.63.3

这版先解决你截图里那两件：翻页效果、页面只占半屏。另外两件（管理任务并行、拖动后刷新）我下一版做，免得一锅端讲不清。

## 一、翻页的卷曲效果——找回来了
之前那个软软的卷曲翻页是 StPageFlip 做的，我前阵子把它停了，因为它在宽屏老摊成左右两页。但其实那是**旧代码被缓存**在作怪（已经修好了）。所以这版把它**重新启用**，并加了两道保险，保证不会再变两页：
1. 把书的容器锁成"单页宽、居中"，宽度上塞不下两页；
2. 建好后再自动检查一次朝向，万一还是两页，立刻退回我自己写的单页翻折。

也就是说：**正常情况你会看到卷曲翻页回来了**；万一哪台机器还出岔子，最差也是单页翻折，绝不会再回到两列。

> 翻不回去那次，多半是你停在第 1 页（第 1 页没有上一页，点左边当然没反应）。换到中间某页，点书页左 1/3 试试往前翻。

## 二、页面只占半屏——改成跟着屏幕走
页宽以前写死 680，大屏上自然显得空。现在改成**按窗口宽度自适应**：约取窗口的 66%，夹在 700~1040 之间（太长一行不利于读）。
- 你这种宽屏：约 1040 宽，比原来 680 宽了一截、留白小了；
- 拉大/缩小窗口会自动重排。

要是你觉得还想更宽、或反而想窄一点，告诉我一个数，我给你加个可调的滑块。
