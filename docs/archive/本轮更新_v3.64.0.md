## 这版新做的：同名书笔记互通
这就是你说的「中途发现更全的版本，想让笔记互通」。在「管理」里新增一张卡：**同名书笔记互通**。

- 它把同一本书的不同版本认到一组——**书名带「精校/全本/番外」这种也能认**（比如「小蘑菇」和「《小蘑菇》（精校版全本+番外完）」算同一本）；
- 然后把你已经写过的那版里的**划线/笔记/书签/简评/评分/收藏/已读/进度**，复制给同组里**还空白**的版本；
- **两个版本都留着**，只往空的补、不动已经有数据的；执行前自动备份。
- 划线/进度按字符位置走——更全的版本开头一般跟旧版一样，所以基本对得上；个别对不上的那条只是不显示，不会错乱。

用法：管理页找到「同名书笔记互通」→ 先「预览可互通的」看一眼哪些书会接上 → 没问题再「执行互通」。

> 注意：它只把数据补给**完全空白**的版本。如果你那本"更全的版本"已经自己写过一点东西，它就不动它（怕覆盖/重复）。
