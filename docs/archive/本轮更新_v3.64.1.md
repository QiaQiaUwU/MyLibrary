# 更新说明 v3.64.1

这版先改你说的「收藏翻页硬挺挺」，另外两件（拖动刷新、手机卡）我下面要你帮我定位一下，免得又白改。

## 收藏翻页——改成真的像翻书
之前那个是整页绕中间转、还会甩过头，所以发硬。现在改成**以左边书脊为轴**翻：页面转到立起来（侧着、看不见）的那一刻才换内容，再落下来——就是翻一页纸的感觉，配了页脚加深的阴影。左右点箭头、键盘 ← →、手机左右滑都触发它。

## 顺手修了一个：拖书去收藏不刷新
拖到「收藏」后，现在会立刻刷新（加红心、必要时进收藏视图），并写进本地缓存，不会被后台再加载时覆盖回去。
