# 更新说明 v3.64.4

## 柔软翻页——查到为什么一直没有了
两个原因叠一起：
1. 那个软卷曲是 **StPageFlip 库** 做的，但它只从 CDN（unpkg）加载，而 unpkg 在国内基本连不上；本地版又一直没放（frontend/vendor/ 里只有说明文件）。所以库根本没加载，一直在用内置翻页。
2. 更隐蔽的：那几个库脚本是 `defer`，会**晚于** `READER_INIT()` 执行——等于初始化时库还没就绪，哪怕连上了也用不上。

这版两头改：
- **内置翻页改柔和了**：透视拉近(2000→1500)、缓动更软更长、卷起的那页加了顺光/背光渐变，看着更像卷纸而不是硬板转。不依赖任何库，人人都有。
- **修了加载顺序**：本地库改成在 reader.js 之前、非 defer 加载。这样**只要你把库文件放进去，真·卷曲翻页就能用了**（以前放了也没用，因为 defer 太晚）。

### 想要"完整版"软卷曲（可选，一次性）
下载 `page-flip.browser.js`（2.0.7 版）放到 `D:\MyLibrary_v3\frontend\vendor\` 下，重启即可。文件可从 jsdelivr/unpkg 任一下载：
`https://cdn.jsdelivr.net/npm/page-flip@2.0.7/dist/js/page-flip.browser.js`
放进去后软卷曲就是离线常驻的，不再依赖网络。我也加了 jsdelivr 作 CDN 备选，联网时也可能自动拿到。
