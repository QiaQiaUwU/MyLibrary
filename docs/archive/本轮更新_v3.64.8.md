# 更新说明 v3.64.8 — 修卷曲翻页的"两页/放大空白/小窗翻不动"

你诊断对了，就是适配（尺寸）没做好。根因：StPageFlip 之前设成 `size:'stretch'`（拉伸铺满容器）——这模式下它**自己按容器大小决定显示一页还是两页**，所以宽屏/放大会摊成两页；而且窗口一变大，旧的画布尺寸没跟上，就**整页空白**。

这版把它改成 **`size:'fixed'`（固定单页尺寸）+ `usePortrait` 强制竖排单页**：
- **永远单页**，不再随窗口宽度摊成两页；
- 窗口大小变了会**按新尺寸重建**（之前已有 resize 重排，这下重建是确定性的，不再放大空白）；
- 小窗、大窗都按当前尺寸生成单页，点左/右各 28% 翻页。
