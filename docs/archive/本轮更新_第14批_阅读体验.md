# 本轮更新（第 14 批）— 进度可见、目录统一生成、划线笔记合并、Quill 读上下文

逐条回应 Point 13–16：

## 13. 阅读时看不到进度 ✅
阅读页加了**始终可见的进度**：
- 屏幕**最顶部一条细进度条**（读到哪填到哪）；
- **右上角一个百分比小标**（翻页模式还会显示 “12/180” 这样的页码）。
- 滚动和翻页两种模式都会实时更新，工具栏收起也还在。

## 14. 有的书没有目录 → 统一生成 ✅
目录改成**统一生成**，分三层兜底，保证任何书都有可跳转的目录：
1. 先按多种章节格式识别：第X章/节/回/卷/部/集/篇、序/序章/楔子/引子/尾声/终章/番外/外传/后记/附录、`Chapter`、`卷一`、`（一）`、`1. 标题` / `1、标题`…
2. 识别不到 → 找“**标题型短行**”（前后是空行、本行很短、不像完整句子）。
3. 还找不到（纯流水账长文）→ **按长度等分**成“第 N 段 · 约百分之几”，至少能跳转。
> AI 辅助找章节：Quill 本来就能在书里搜索/定位，需要时可以让它帮你找“第X章在哪”。要不要再做个“一键 AI 重建目录”按钮，你说一声。

## 15. 笔记/划线/书签合并 + 笔记可改 ✅
按你说的“直接做一起”：
- **划线 = 收藏句子**。选中正文 → 选颜色就划好了。
- **划完可选填笔记，也可以不写**（选择菜单里的“笔记”＝划线并写一句；只点颜色＝只划线不写）。
- 列表里（“**划线·笔记**”页）每条都能：**点句子跳到原文**、**写笔记/改笔记**（这下能改了）、**删除**。
- 不再“划线存一份、笔记又存一份”那种重复；原来的“笔记”单独页并入了“划线·笔记”。
- （Quill 帮你存的笔记仍能在 书房→收藏·手账本 里看到。）

## 16. 阅读助手能读上下文聊情节 ✅
其实这块后端本来就在做——我把它**加强并加宽**了：
- Quill 每次都会带上你**当前位置前后的原文**（窗口从前 1500/后 500 字 **加宽到前 3000/后 1500 字**）。
- 提示词里明确告诉它：**优先结合这段原文跟你讨论此处的情节、人物、伏笔、写法**；超出这段的内容再自动调“前情提要 / 全书搜索 / 角色追踪”工具。
- 所以你读到哪，直接问它“这里他为什么这么做 / 前面那个人是谁 / 这段在铺垫什么”，它能接着聊。
