# 本轮更新（第 20 批）— 笔记跳转、总评/简评、朋友圈书单分享

## 1. 笔记入口可跳手账本 ✅
书详情里那行「笔记 …共 N 条」现在可点——点一下直接跳到**总览→收藏的手账本**，并自动翻到这本书那一页，看全部划线/笔记。

## 2. 总评 / 简评 / 备注 ✅
书详情里在「评分」下面加了一栏「**总评**」：点一下就能写这本书的总体看法 / 简评 / 备注（失焦自动保存）。它和评分都放在一起，都属于"我的评价"。这条简评也会用在下面的朋友圈书单里。

## 3. 朋友圈书单分享 ✅
在 **阅读历程 → 回顾** 右上角加了「**分享书单**」按钮：
- 自动把你**读过的书**（读完 / 已读 / 打过分 / 写过总评的）列成一张卡片。
- 可**筛选**：按作者、按标签、按时间（今年/全部）。
- 卡片里每本显示 书名 · 作者 · **星级** · **简评**。
- 可**自定义**：改书单标题、勾选隐藏星级、隐藏简评、换 3 种配色（纸色 / 夜墨 / 青竹）。
- 「**下载书单图片**」导出成一张图（SVG），可直接发图或截图发朋友圈。

> 这版先做了筛选 + 星级简评 + 配色 + 下载。你提到的"自定义上传背景图、拖拽调整排版"属于更进阶的编辑，下一轮再加（要的话告诉我优先级）。
