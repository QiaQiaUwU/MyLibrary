# 本轮更新（第 39 批）

## 1. Quill/伴读连不上 —— 修好了（关键）
原因：你**测试成功**后那个「待保存」标记是在提醒你还得点一次「保存配置」；而"存成配置档"只是存档，
并没把它设为当前接口。于是 Quill 读到的还是空 key，就一直说"需要配置 API key"。

修复：
- **测试连接成功 = 直接保存为当前接口**（不用再点保存），Quill 立刻能用，提示也会变成"已连接并保存"。
- **"存成配置档" = 同时设为当前接口**（存了就生效）。
- 点配置档小药丸（如 claude4.6）激活，本来就会写进当前接口——所以你现在重新点一下「测试」、
  或点一下「claude4.6」那个配置档，Quill 就连上了。
（伴读/书库问答都是每次实时读最新配置，保存后即时生效，不用重启。）

## 2. 阅读时双击 Quill 快速切模型
- 阅读器里**双击右下角 Quill**，弹出一个"切换模型"小窗，列出你存的配置档，点一下即切（长按拖动照旧）。
