# 本轮更新（第 41 批）— AI 配置 UI 按你的意见改

## 1. 测试 与 保存 分开
- **测试**：只是测试连接，成功就显示「✓ 连接成功」，**不再自动保存**。
- **保存配置**：单独一步——点了才让 Quill 用上并记住（下次不用重输 API、重选模型）。
- 配置档（小药丸）：**存档只是记住**，点小药丸才切换使用；存档不再自动激活。
这样"测试是测试、保存是保存"，互不绑定。

## 2. 模型只用一个口
- 去掉了单独的「获取模型列表」按钮和下面那个独立下拉。
- 现在就一个「模型」输入框：**点开它会自动拉取可选模型**做成下拉补全，直接选；也能手填。一个口搞定。
