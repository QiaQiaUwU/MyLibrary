# 状态总览（第 46–47 批做了什么 · 还有什么没做）

## ✅ 这两批已做（都过了编译/JS 校验，关键逻辑有单测）

### A. 简评/总评丢失 —— 找到真因并找回
- **真因**：详情弹窗的「总评」只从内存书单读取，而**从搜索结果点开的书不带 `user_notes`**，于是每次重开显示空白，看着像没保存，其实一直在库里。
- 现在弹窗打开会**拉服务端权威记录**（`/api/book?id=`），总评/评分/状态总是最新——**你以前写的简评会重新显示出来**。
- 保存改稳：用 `findBook`（搜索结果也更新）、`await` 保存、失败弹提示；**关弹窗前若还在编辑会先存一次**，不再因没失焦就关而丢。

### B. 总评放到阅读记录最上面
- 新增 `/api/journey/reviews`，汇总所有非空总评（按最近阅读排序）。
- 「阅读历程 → 总览」**最上面新增「我的总评」区**，每条显示 书名·作者·星级·简评，点开那本书。也是集中找回旧简评的地方。

### C. 书架最新在读放最前
- 阅读保存进度 / 标在读会盖 `last_open`；书架「在读」「已读」两排**按最近阅读倒序**。

### D. 阅读器划线大改（之前最卡手）
- 定位改用**全书字符偏移**（不再用滚动位置）：跳转精准、同句多次不跳错、老划线自动找回位置。
- 加划线**只就地重绘**，不再整篇重渲染 → 不再"跳到很后面"、不卡。
- **长段/跨自然段也能划**了。
- 做笔记**沿用上次选的颜色**；划线列表里每条有**换色小圆点**，直接改色不用删了重画。
- 划线菜单**浮在选区旁边**。

### E. 翻页 & 适配
- **往前翻也有动画**；手机/窄屏改用贴合可视区的单页翻 + 修分页几何（修「下面一半没字」）。

### F. 搜索与筛选分开 + 多选栏看得清
- 搜索框只搜索；**筛选漏斗移到下面独立一行**，点一下展开/收起整块。
- **多选栏固定深色底**（修你那"白底看不清"），退出做成明确按钮；**手机上占满底部、可横滑、退出常驻右端**不会被挤出屏幕。

### G. 移动端"加载不全" —— 修了
- 之前后台补齐剩余书后，**只有在书房页且停在顶部才重排**；在书架页或已滚动时新书不出现，看着像没加载全。现在**后台取完总是重排**（保留滚动位置、不打断打字搜索）。

### H. "本地网站快捷入口放到屏幕"
- 「安装到桌面/主屏」入口**默认就显示**了（以前藏起来等浏览器主动弹，很多浏览器不弹＝等于没有）；点了若浏览器没给一键安装，会按 **iOS / 安卓 / 电脑** 给清楚的"添加到主屏/桌面"步骤。
- 想要纯本地、像独立软件那种：仍是双击桌面快捷方式（pythonw 独立窗口），这条之前已做。
