## 2. 拖拽导入书籍 ✅
- 把 **txt / epub / md / html** 文件**拖到书房页面**，会弹出覆盖提示，松手即导入。
- 文件存进 `书库/books/_import/`，并登记入库（书名暂取文件名）。
- 这是**基础导入**：不做精细去重和元数据精提取——导入后到管理页跑一下「去重」「AI 打标签」即可。

## 3. 版本管理页（重复副本）✅
- 管理页新增「**版本管理（重复副本）**」卡：按保留的主版本，列出去重时被隔离的副本（含原路径、判定原因）。
- 每个副本可「**找回**」：把它从隔离区拷回 `books/_restored/` 并重新入库（万一去重误杀了想要的版本）。

## 4. 备份信息表 导出 / 导入 ✅
- 管理页新增「**备份信息表（导出 / 导入）**」卡：
  - **导出 JSON**：把全库的 书名 / 作者 / 题材标签 / 评分 / 阅读状态 / 笔记 导出成一个 JSON，做备份。
  - **导入 JSON / .mrpro**：重装或换库后导入回填——按 **书名+作者** 或 **原文件名** 匹配现有书，回填评分/状态/收藏/笔记 + 题材标签。
  - 也能直接导入 **Moon+ Reader 的 .mrpro**（自动读取里面的 31.tag 信息表，把书名/作者/category 标签导进来）。
- 说明：只**读** .mrpro、不**回写** .mrpro（从零造一个 Moon Reader 备份结构风险太大、也没法验证）；日常备份用本工具导出的 JSON 就够。
