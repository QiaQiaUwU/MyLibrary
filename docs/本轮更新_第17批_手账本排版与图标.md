# 本轮更新（第 17 批）— 手账本排版 + 桌面图标换成 Quill 羽毛笔

## 1. 手账本：按作品归并、翻页、排版重做 ✅
- **排序**：按你阅读/记录的时间，每个作品占一页，一本书相关的（划线/笔记/和 Quill 的讨论/书签）全在同一页——左右 `‹ ›` 翻页。
- **排版**重做得好看了：
  - 书名做成大号衬线标题，下面一条带强调色的分隔线，作者用「— 作者」小字；
  - 每个小节（收藏的句子 / 笔记 / Quill / 书签）有带图标的标题 + 渐隐分隔线；
  - 每条记录左侧有一条时间线带圆点，划线用衬线大字、笔记放在浅底小卡里、时间很轻；
  - 整页是一张带阴影的纸，翻页有翻书动画。

## 2. 桌面快捷方式图标 = Quill 羽毛笔 ✅
按你说的，桌面图标直接用 Quill 那只**羽毛笔**：我把 App 里同一条羽毛笔轮廓画进了一个棕色圆角底的 `app.ico`（16~256 多尺寸都清晰）。`创建桌面快捷方式.bat` 本来就指向 `app.ico`，所以新建的快捷方式就是这只羽毛笔。
> 提醒：如果你之前已经建过快捷方式、桌面还显示旧图标，那是 Windows 把图标缓存了——**重新跑一次「创建桌面快捷方式.bat」**（会覆盖），或右键快捷方式→属性→更改图标→选本目录的 `app.ico`，就更新了。

覆盖后**强刷两次**。手账本在 总览 → 收藏 里打开。
