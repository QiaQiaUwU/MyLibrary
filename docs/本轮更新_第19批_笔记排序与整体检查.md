# 本轮更新（第 19 批）— 笔记排序/排版 + 阅读里程；附整体检查清单

## 本轮改的

1. **去掉笔记前的 📝**：划线列表里笔记不再带 emoji，靠排版区分（笔记是浅底小卡，划线是衬线引文）。
2. **笔记按"在书里的先后顺序"排，不按记录时间**：阅读器的「划线·笔记」列表、手账本里每本书的划线/笔记/书签，都按它们在原文中的位置从前往后排——像纸质书一样固定不变。
3. **时间仍然记录**：每条记录下面照样显示记录时间（只是不拿它来排序）。
4. **手账本每本书显示"阅读里程"**：在总览→收藏的手账本里，每本书页顶部多了一条：`X/X 开始读 · 读到 Y% · 最近 X/X · 共 Z 分钟`（数据来自阅读进度和每日阅读记录；没读过的书不显示）。

## 你问的"桌面快捷启动 bat 有没有做"
做了，在包里：
- `创建桌面快捷方式.bat`：双击它会在桌面建一个 **MyLibrary 快捷方式**（图标就是 Quill 那只羽毛笔 `app.ico`），直接跑 `pythonw main.py`、无黑窗、不经过任何 bat。
- `启动.bat` / `MyLibrary.bat`：双击直接启动（已全改成纯英文 ASCII，不再有中文乱码问题）。
