# 本轮更新（第 26 批）

## 1. 阅读器·翻页终于能点了（重点）
之前 StPageFlip 把书页上的点击「吞掉」了，鼠标点哪都翻不动。改成**捕获阶段**接管点击：
点书页左 28% = 上一页、右 28% = 下一页、中间 = 收/放工具栏，点哪都生效。

## 2. 阅读器·那个碍事的 ⋮ 圆圈去掉了
点正文中间就能收/放工具栏，不再需要那个悬浮圆圈，已隐藏。

## 3. 阅读器·声音能关了 + 小窗能拖
- 环境音面板顶部加了显眼的「全部停止」按钮（一键静音，不用再往下翻找）；点声音名字本来就能单独停。
- 环境音小窗：**抓住顶栏可以四处拖动**。
- Quill 悬浮球：**可以四处拖动**（轻点照常打开 Quill，拖动不会误触）。

## 4. 阅读器·按钮改名
划线条目的「改笔记」改成「编辑」。

## 5. 花园·长按改为进入编辑
- 「批量编辑」改名「编辑」。
- **长按一棵树 = 进入编辑（多选）模式并选中它**，不再弹"要不要移除"的提示。

## 6. 设置·API 配置精简
- 去掉「名称」一栏（不用给接口起名了）。
- 按钮「测试连接」改名「测试」。
- 连上后，**「测试」按钮本身变成「✓ 连接成功」**（绿色），不再在上面显示一长串；改了 URL/Key/模型会自动复位成「测试」。

## 7. 找回作者·不再把题材/格式当成作者
np、nph、txt、_1、无名、未知、匿名、3p 这类**题材标签/文件格式/垃圾词**一律不再当作者
（np 是题材不是作者）。真作者（诗无茶、淮上等）照常找回。

## 8. 启动·打开的是"应用窗口"而不是浏览器标签页 + 图标重做
- 启动后优先用 Edge/Chrome 的 **--app 独立窗口**打开（无标签栏/地址栏，跟你之前"安装的应用"一样）；
  找不到这两个浏览器才退回默认浏览器开标签页。
- 桌面快捷方式：双击 `创建桌面快捷方式.bat` 生成，目标就是启动器，**图标换成清晰的羽毛笔**
  （不再像贝壳/咖啡豆）。若已生成过旧快捷方式，删掉重新跑一次 bat 即可。
