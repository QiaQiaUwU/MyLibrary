# 本轮更新（第 27 批）

## 1. 搜索 & 筛选
- **关键词搜索修好了**：以前只在「前端已加载的那部分书」里找，13万本根本搜不全。现在搜书名/作者是**整库搜索（走服务端）**，输 "青序" 这种能在全部藏书里命中。
- **筛选折叠修好了**：之前点左边那个箭头没反应——是代码找错了元素 id。现在**点箭头或点"按标签筛选"几个字**都能把整块筛选收起/展开。
- **搜索历史**：搜过的词会记下来（存在你本机浏览器里）。搜索框聚焦且为空时显示「最近搜索」，**每条可单独点 × 删除**，也能「清空」。

## 2. 阅读状态：未读 / 在读 / 已读 三态
- 书籍详情里的「状态」变成一个**可点的按钮**：点一下循环 未读 → 在读 → 已读。
- **打开阅读器就自动标「在读」**，这本书会出现在书架的「在读」一排（已读的不会被降级回在读）。
- 书架里新增「**已读**」一排（和「在读」一排并列）。
- 标「已读」的时刻就记为读完时间（用的是 reading_status=finished，已写回数据库）。

## 3. 自定义标签进标签库
- 在「编辑书籍信息」里手输回车加的标签，**会进标签库**，下次编辑别的书时作为建议出现（存在你本机浏览器里，最多记 300 个）。
