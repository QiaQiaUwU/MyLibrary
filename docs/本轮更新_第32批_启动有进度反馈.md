# 本轮更新（第 32 批）— 启动时能看到进度，独立窗口照常弹

## 问题
你用的是 pythonw 那个"没有黑窗口"的快捷方式 + 我又加了"等服务起来再开窗口"，
结果点了之后有 1~2 秒**什么都看不到**（没有黑窗、窗口还没弹），就以为"点不开了"，也不知道进度。

## 这批怎么改
- **推荐用 `启动.bat` 启动**：它会弹一个黑色小窗，立刻显示
  「MyLibrary is starting … 独立窗口稍候弹出 …」，服务好了会显示「✓ 服务就绪，正在打开独立应用窗口」。
  这个黑窗就是服务本身（**别关，关了＝退出**）；真正的界面是另外那个**独立窗口**。
  这样你既有"加载进度"可看，又是独立窗口。
- 服务在等就绪/打开窗口时，会往这个黑窗里打字，所以你能看到它在干活、有没有报错。

## 怎么用（最稳）
1. 用 v35 覆盖，先在任务管理器结束所有 python.exe / pythonw.exe。
2. 双击 **启动.bat**。黑窗出现 → 等几秒 → 独立窗口自动弹出。
3. 想让桌面也有入口：右键 `启动.bat` → 发送到 → 桌面快捷方式（再右键改图标为 app.ico）。

> 如果黑窗里出现红色报错，截图发我——那就是真正的原因（多半是书库硬盘/路径）。
> 等彻底正常了，想要"无黑窗"再换回 pythonw 那个快捷方式也行。
