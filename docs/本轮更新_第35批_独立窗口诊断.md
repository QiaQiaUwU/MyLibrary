# 本轮更新（第 35 批）— 独立窗口：用快捷方式，不要手敲网址

## 先说清楚你截图里看到的
那个在你**平时的 Edge 里、和一堆别的标签并排**的「书房·MyLibrary」标签页，是因为
**网址被在你日常 Edge 里打开了**（你手动敲 127.0.0.1:8765，或地址栏历史点进去的）。
服务这次是好的（进度条转起来了 = uvicorn 崩溃已修复）。

桌面快捷方式打开的"独立窗口"是**另开的一个单独窗口**：没有标签栏、没有地址栏，
而且用的是一个空白配置（不会出现在你日常 Edge 的标签里）。所以要的是"用快捷方式启动"，
而不是在浏览器里敲网址。

## 这批加了什么
- 启动器现在会把"到底开没开成独立窗口"写进根目录 `mylib_log.txt`：
  - 成功：`打开独立应用窗口：…msedge.exe --app=…`
  - 没找到浏览器：`⚠ 没找到 Edge/Chrome，退回默认浏览器（会是标签页…）`
  - 出错：`⚠ 打开独立窗口失败…`
  这样万一还是标签页，看一眼日志就知道原因。

## 怎么得到独立窗口
1. **双击桌面的 MyLibrary 快捷方式**（不要在浏览器里敲网址）。
2. 会**单独弹出一个窗口**（无标签、无地址栏）；它不会混在你日常 Edge 的标签里。
3. 如果弹出来的还是标签页，打开根目录 `mylib_log.txt`，把最后几行发我。

> 备选（最稳的独立 App）：在书房页面里点 Edge 地址栏右侧的"安装此应用"图标，
> 装成 PWA，会有独立窗口 + 自带羽毛笔图标的快捷方式；以后从它打开即可（仍需服务在跑）。
