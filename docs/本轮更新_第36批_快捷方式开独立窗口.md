# 本轮更新（第 36 批）— 让快捷方式直接开"独立应用窗口"

## 你的诉求（这下清楚了）
点桌面快捷方式 → 直接弹出像截图那样的**独立应用窗口**（无标签栏/地址栏），
而**不是**在浏览器里开个标签页。事实上之前点快捷方式并没有独立开窗口——
因为启动器没找到 Edge，退回了"默认浏览器"（于是成了你日常 Edge 里的标签页）。

## 修复
用**注册表 App Paths**（Windows 登记的真实安装路径）来找 Edge/Chrome，最可靠；
找不到再猜常见目录、再看 PATH。找到后用 `--app` 开**独立窗口**。
这样快捷方式点下去就直接是独立应用窗口，而不是浏览器标签。

启动器还会把结果写进根目录 `mylib_log.txt`：
- `打开独立应用窗口：…msedge.exe --app=…` = 成功开了独立窗口；
- `⚠ 没找到 Edge/Chrome，退回默认浏览器…` = 没找到（这才会变标签页）。

## 用法
1. v38→用 **v39** 覆盖；先在任务管理器结束所有 python.exe / pythonw.exe。
2. 双击桌面 **MyLibrary** 快捷方式（别在浏览器里敲网址）。
3. 应该**单独弹一个窗口**（无标签、无地址栏）。若还是标签页，把 `mylib_log.txt` 最后几行发我。

> 那个独立窗口用的是一个专属的空白配置（所以不会混进你日常 Edge 的标签）；本地用不需要登录，放心。
