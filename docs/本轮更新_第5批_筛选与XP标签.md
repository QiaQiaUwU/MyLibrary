# 本轮更新（第 5 批）— 晋江式标签筛选、鼠标浮动、XP 标签、参考 AO3

## 1. 标签筛选改成晋江那种（藏进搜索框、分类、可排雷）
我去看了晋江和 AO3 的筛选是怎么做的：
- **晋江**：标签按维度分类，每个标签能选「必含 / 可含 / **不含（排雷）**」，手机版就是点标签上的「＋加入包含 / －加入排除」。
- **AO3**：标签分 Fandom / 人物 / 关系 / **附加(Freeform)** 四类，Freeform 里就是各种 trope、情绪、设定、XP（Slow Burn、Pining、Possessive、BDSM…）。

照这个做法，本项目的筛选改成：
- **藏在搜索框里**：点一下搜索框（或右边的漏斗按钮）才展开筛选面板，不点就收着，不挡书架。
- **标签按维度分组**，不再堆成一坨：性向/CP、世界观/背景、设定/套路、人设、情节/关系、**XP/取向**、**尺度**、结局、篇幅、其他。
- 每个标签**点一下＝必含（绿色＋）**，**再点＝排除/排雷（红色－划掉）**，**再点＝取消**。可以多选叠加。
- **按作者、按题材两种分类下都能用**（之前只在题材下，按你最新说的改成都能用）。
- 面板只列**库里真实存在**的标签并带数量；点「完成」或点面板外面收起。

## 2. 鼠标浮到书上
- 书会**轻微放大**表示被选中/聚焦。
- 浮出这本书的**标签**（封面底部一排小胶囊）；同时鼠标悬停也有系统 tooltip（书名 · 标签）。

## 3. XP 词汇 +（去掉「双洁」）
- 按你说的**删掉了「双洁」**。
- 参考 AO3 的 Freeform 和晋江常见 trope，加了 **XP/取向** 维度：强强、强制、占有欲、病娇、黑化、偏执、调教、制服、囚禁、主从、双性、信息素、标记、失忆、替身、钓系、禁欲系、高岭之花、白切黑、互宠、双向救赎、强取豪夺、死遁、苏爽、甜虐交加、微虐。
- 另加 **尺度** 维度：清水 / 微H / 高H（像 AO3 的分级，方便排雷或专挑）。
- 标签总数现在 **153 个、10 个维度**。打标签提示词也会让 AI 跨维度多打几个。

> 注意：词库扩充影响的是**“以后打标签”**。要让这些新标签出现在书上、能筛，需要在管理页对书**重跑一次 AI 打标签**（或在书详情手动加）。已有的旧标签不受影响、照常能筛。
