# 本轮更新（第 6 批）— XP 改成“玩法”、书桌装扮、主页自定义、设置小窗

## 1. XP 维度修正：现在是真正的“玩法”
你说得对，上一版我把 trope 混进 XP 了。这版把 **XP/玩法** 单独拎出来，放真正的玩法（性癖）标签：
> 强制、半推半就、调教、主奴、束缚、道具、制服诱惑、角色扮演、责打、囚禁、露出、多人、双性、生子、产乳、信息素、标记、强制发情、触手、窒息、言语羞辱

原来错放在 XP 里的那些 trope 已经各回各家：
- **人设**：病娇、偏执、黑化、钓系、禁欲系、高岭之花、白切黑
- **情节/关系**：占有欲、强取豪夺、死遁、失忆、替身、双向救赎、互宠
- **风格/基调**：苏爽、甜虐交加、微虐

另外保留 **尺度** 维度：清水 / 微H / 高H（像 AO3 的分级，方便挑或排雷）。
现在共 **165 个标签、10 个维度**。打标签提示词也加了一条：**XP/尺度 开头一般看不出，没明确证据就别瞎标，这类主要靠你手动加**（在书详情「编辑」里加）。

> 提醒同上：新词库影响的是“以后打标签”。要让新标签出现在书上并能筛，得在管理页**重跑一次 AI 打标签**，或手动加。

## 2. 书桌装扮（阅读页两侧）做了
- 在阅读页正文**两侧的留白处**可以放自定义图（左/右各一张）。
- 图片向内**羽化**过渡、半透明，**不挡正文、不可点**，只做氛围。
- 在哪设置：书房页右上「主题」弹窗里，新增「**书桌装扮**」一栏，传左侧/右侧图即可；阅读页下次打开就生效。

## 3. 主页自定义也补齐了
- 整页背景 / 顶部插画条 / 四角装饰 的上传**本来就在**「主题」弹窗里（之前可能被缓存挡住没看到，强刷一下就有）。
- 这版新增 **「背景浓淡」滑杆**：自定义背景太抢眼影响看书，就往「朦胧」拉；想看清背景就往「清晰」拉。滑杆状态记住，下次进来自动应用。

## 4. 设置改成“小窗弹出”，而且灵活
- 点右上「设置」，现在在一个**居中的小窗**里打开，而不是整页跳走，看完关掉就回到书房。
- 这个小窗**可缩放**（拖边角改大小）、带滚动条；要是浏览器拦截弹窗，会自动退回整页打开。
- 内部用了个通用 `popout()`，以后别的面板要弹小窗也能直接复用。
